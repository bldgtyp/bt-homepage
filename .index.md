# bt-homepage — source for bldgtyp.com

Static public website for BLDGTYP, LLC. Hugo is the source of truth. Only
generated `site/` output deploys (Dreamhost via GitHub Actions on push to
`master`). `research/` is supporting material, not deploy output.

Authoritative repo/deploy/editing rules live in `CLAUDE.md` (read it before
editing) and `README.md`. This file is only a navigation map.

## Contents

### `site/` — deployable static site
- `index.html`, `about/`, `learn/`, `ma/`, `nyc-thermal-bridging/` — generated by Hugo. Edit `hugo/` sources and rebuild, don't hand-edit generated HTML.
- `assets/` — PNG imagery (`hero-*`, `detail-*`, `who-*`, service thumbnails), `favicon.svg`, plus `full-size/` (8 images) and `ph-logos/`.

### `hugo/` — Hugo source for all pages
- `hugo.toml` — publishes into `../site`. Expected: hugo v0.124.1+extended.
- `content/` — `_index.md`, `about.md`, `ma.md`, `nyc-thermal-bridging.md`, `learn/_index.md`.
- `data/videos.yaml` — video library feeding `/learn/`; grouped by topic (each video: id, title, host, year). Source of truth noted in file header.
- `layouts/` — `_default/` (baseof, single), `index.html`, `learn/list.html`, `partials/` (nav, footer, site-styles, site-scripts).

### `research/` — source research + synthesis (not deployed)
- `episode-notes/` and `transcripts/` — ~36 podcast-guest files each (one per interviewee, plus `trailer.md`), parallel filenames.
- `synthesis/` — homepage positioning/copy outputs: `homepage-alignment-and-business-model.md`, `homepage-copy-redline.md`, `positioning-decisions.md`, `marketing-rules.md`/`.html`.
- `audio-summary/` — 3 `.m4a` audio summaries.
- `scrape_transcripts.py` — transcript scraper.

## Notes
- `CLAUDE.md` is symlinked from `AGENTS.md`.
- Deploy mirrors `site/` with `--only-newer`, no delete flags — server may hold content not in this repo. Do not add delete semantics.
- Ignored: `.DS_Store`, `hugo/.hugo_build.lock`, `hugo/resources/`.
